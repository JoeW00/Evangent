# 爭戰與建造

當我們與神一起在建造家的時候，要記得祂才是建築師，我們都只是承包商。此外也必須理解我們不只是在與神一起建造自己的家，同時也在重新打造出一個家本來該有的樣子。

每當我想到重新建造，我總是立刻會想到尼希米記。尼希米一聽說耶路撒冷毀了，他立刻有所行動（見尼希米記二到三章），他把以色列人招聚在一起來重建城牆。當你在讀這個故事時，你會看到許多人都是在重建自己家門口的那段城牆。在他們盡力把自己家附近的城牆蓋好的同時，他們也扮演了另外一個更重要的角色，就是把整座城給保護好。為了要使以色列可以成為耶和華定意她要成為的大國，神是從一個家開始的。在地圖上八字都還沒有一撇之前，他們就已經先成為兒女、是屬神的百姓。我們不光是加入神的行列一起打造自己的家，我們也是在按著神起初的心意在重建家庭。

在尼希米著手重建前，有些人聽說了他打算要做的事，那讓他們有點不開心。事實上，他們「聽見有人來為以色列人求好處，就甚惱怒。」（尼希米記二章10節）當我們說要重建家庭的時候，我們必須曉得仇敵絕對不會因此而感到開心。畢竟牠長久以來一直都想要毀掉家庭，因此當你要去重建自己家的那個部分時，牠絕對不會樂見其成。仇敵當年想要藉由恐嚇和謊言來使百姓無法專心地去重建，但是尼希米不僅看穿仇敵的詭計，同時也把對的人放在對的位置上，好讓重建計劃得以順利進行下去，仇敵的計畫也以失敗告終（見尼希米記第四章）。

雖然有必要知道仇敵不會開心，但更重要的是我們必須持續認出是神在保護我們。尼希米這個字的希伯來文原意是「耶和華是安慰」，而他的確也描繪出了我們的安慰者—聖靈的樣式。或許仇敵會想方盡法地攔阻你建造你的家，但是牠完全不是聖靈的對手。聖靈會把對的人和天使放在對的地方，來守護好你的家。「修造城牆的，扛擡材料的，都一手做工一手拿兵器。修造的人都腰間佩刀修造，吹角的人在我旁邊。」（尼希米記四章17-18節）神已賜給你能力，讓你可以同時一邊爭戰、一邊建造。

筵席已經擺設好了

這些年來父母們曾問過我許許多多的問題，不過我最常被問到的就是有關屬靈爭戰。我想帶各位讀幾節經文，相信會有助於你知道如何同時爭戰與建造。其中最基本的是：「在我敵人面前，祢為我擺設筵席；祢用油膏了我的頭，使我的福杯滿溢。」（詩篇二十三篇5節）只要一看到仇敵出現，就要曉得神早已擺設好筵席。早在我們都還沒起來爭戰之前，祂就已經在邀請我們要成為這場筵席的座上賓。順序應該是先拿起這餐桌上的刀叉，然後才去看要用什麼武器爭戰。畢竟如果沒有吃好喝足，人們恐怕難以好好應戰。當我們好好安息並且以神的信實為糧，祂就會以油來膏我們的頭，祂的膏油將會把軛給撐斷並挪去一切重擔（見以賽亞書十章27節）。

現在讓我們以一個有被好好餵養和得以安息的角度來看下面這段經文：「他們當將榮耀歸給耶和華，在海島中傳揚祂的頌讚。耶和華必像勇士出去，必像戰士激動熱心，要喊叫，大聲吶喊，要用大力攻擊仇敵。」（以賽亞書四十二章12-13節）這裡的海島指的就是領土的邊境，也就是地的疆界。換言之，只要有頌讚的聲音揚起，天國就會啟動邊境的巡航，如果有神這位萬軍之耶和華在防守我們所在之地，敵軍肯定是難以入侵。

我會想像這個畫面就好像是，神放了一艘潛水艇潛伏在海面下，只要敵軍的艦艇膽敢輕舉妄動構成威脅，祂隨時伺機預備好發射魚雷。我知道當這節經文被寫下來的時候還沒有潛水艇或是魚雷。不過這幅圖畫確實很容易幫助我們明白神如何以祂的大能在為我們家庭爭戰。從這節經文，我們也可以看見就算仇敵嘗試想要我們把焦點轉向牠自己，但是只要我們專注仰望和愛慕主，就能帶出極大的能力。當仇敵不斷地哀求要我們關注牠，但只要我們不予以理會並將專注力放在主身上，祂就會緊迫逼人地盯著仇敵。仇敵想要的是從人而來的關注，牠可不想要神注意到牠，牠雖然極度地渴望我們多多關注牠，但同時也很怕神注意牠。

從詩篇一四九篇我們可以看見聖經裡講到在爭戰中讚美神是多麽具有大能的另一節經文。

願聖民因所得的榮耀高興！願他們在床上歡呼！願他們口中稱讚神為高，手裡有兩刃的刀，為要報復列邦，刑罰萬民。要用鍊子捆他們的君王，用鐵鐐鎖他們的大臣；要在他們身上施行所記錄的審判。祂的聖民都有這榮耀。你們要讚美耶和華！

詩篇一四九篇5-9節

我們能夠藉由讚美來對仇敵施行審判，換句話說，哪個家中充滿讚美，那個家就能夠對仇敵造成傷害。「祢因敵人的緣故，從嬰孩和吃奶的口中，建立了能力，使仇敵和報仇的閉口無言。」（詩篇八篇2節）就算仇敵在我們的營壘外頭吼叫得再大聲，從孩子口中所發出來的讚美聲音就能使牠們閉口無言。

## 是時候了！

當第二集的《超人特攻隊》總算在電影院上映的時候，我實在難掩自己的興奮之情。畢竟這個系列出第一集也已經是十四年前的事了，此刻的我早已等不及想帶全家人一起去看這部新片。我也滿心期待要被第二集的電影給深深地感動，因為我從第一集真的學習到太多東西了。神讓我在《超人特攻隊》的第一集裡看到，即便對於宗教狂熱也不表示就會希望我們能夠活出大能的生命，也不想看見我們活出自己真實的身份。反倒是只想要我們當個正常人，過著平庸的日子（見哥林多前書三章3節）。

第一集快要結束的時候，反派角色辛拉登（Syndrome）打算要欺騙地球上的人，他嘗試要用計讓人以為他有能力拯救他們，但他其實根本就辦不到。他的目的只是想要摧毀掉這些真正的超能英雄並取而代之。而他之所以會這麼做，都是因為他小時候希望有天可以成為像主角一樣的超能英雄，但是反倒卻被對方給傷害了，這聽起來不也挺像我們仇敵的故事。

電影中當超能英雄被告知不可以再使用自己的能力或是讓人知道自己的真實身份，社會大眾基本上也就忘了他們的存在。只有那些超自然家庭才知道事情的真相，並且有能力阻止辛拉登佔領世界。第一集演完之後，就這麽過了漫長的十四年。

在那段期間裡，我清楚地看見神怎麼大大使用那些專門做兒童電影的編劇或是製作團隊，即便他們大部分的人可能自己都不曉得。我猜那或許也不是編劇的初衷，不過神卻不著痕跡地透過這些電影把祂的故事和信息給傳遞了出來。雖然有為數眾多的基督徒不見得同意我的這個觀點，認為這些電影是想要毀了我們的孩子。不過在我看來反倒覺得這更加證明了，神真的可以使用電影來成為傳遞信息的媒介。因為有上百萬名的孩子大概永遠不會踏進教會一步，因此祂就在電影院裡與他們直球對決。

我在一部又一部的電影中看見了福音的影子，幾乎很容易就可以在這些兒童電影裡看到死亡與復活、究竟是出於愛還是恐懼在做決定，又或是神的超自然大能以及基督徒的身份等這類主題。每次只要看了一部這樣的電影，等我走出電影院的時候，我就會感覺神放在我心裡的感動是，即便是在電影院裡看電影的時間祂都仍是在帶下影響和改變。透過這樣的方式，祂能夠在某個特定的時間點傳遞給孩子們一個特定的信息。

祂對我說，我的話語是永恆的，沒有時效性。不論是用DVD或是在線上觀看，這些電影都會對人們傳遞出某個獨特的信息。不過當電影在電影院裡上映的時候，那個信息就好像他們是所領受到的新鮮的糧，你仔細去觀看我在那些時間裡透過這些電影要來對你說什麼。

我相信這是為什麼《超人特攻隊》出了第一集以後，會隔了十四年才又出下一集，因為世人還沒有預備好要來領受續集電影中所蘊藏的信息。

在《超人特攻隊》第二集裡也有許多非常大能的信息；就好像電影一剛開始，這一家人一起吃著晚餐並討論有哪些超能英雄還是不合法也不得表明自己身份時，家中排行老二的小飛（Dash）就大聲嚷嚷著說：「我們想要打擊壞蛋，這定義了我的人格。」我相信這是所有信徒的呼求，對我們的孩子來說更是如此。

那幕結束之後過沒多久，就發生了一連串的事件，發現有人正計劃要幫助讓社會上所有人都可以重新接受超能英雄。這個計畫是希望幫助社會大眾了解，這些超能英雄家族裡的人絕對都是出於善意，以及當這些超自然家庭可以完全按著個人真實身份去活時，不僅能夠保護他們，也會讓社會變得更好。

在這部續集裡，又出現了另外一個名叫「螢幕魔人（Screen-slaver）」的反派角色，是專門騙人的。這次她是利用螢幕進行催眠，把人都變成自己的奴隸。她從頭到尾就是要設計讓這個超自然家庭會照著她的計畫做，包含利用他們的力量來幫自己和毀掉他們的名聲。她在這場戰役當中所使用的武器是螢幕，只要能夠讓人眼睛盯著螢幕，那麼無論她要他們做什麼，他們都會照辦。因此只要她有辦法抓住他們的注意力，她就可以使用他們的能力去完成她所策劃的壞點子。幸好，這家人最終擺脫了她的螢幕控制，並且成功地攔阻了她的計畫。

雖然這部電影還有許多其他的信息可以分享，不過其中一個重點就是，這個世界需要超自然家庭培養出被聖靈所引導的孩子，好讓他們可以被興起好去攔阻仇敵的工作。除了他們，沒有其他人可以辦得到。這世界需要你和你的孩子，你們就是世人心目中最偉大的英雄（當然是僅次於耶穌啦）。世人需要你們擺脫掉一切的綑綁，好讓你們可以心無旁騖，別再順著魔鬼的意思，而是可以好好運用神所賜的能力與身份。

神在助我們眾人一臂之力，好讓我們可以重新照著自己在這個社會中所該站的位置與角色去發揮。祂也在教導我們去發現和管理好祂放在我們孩子身上的恩賜才幹。當別人都找不到方法可以解決眼前的問題時，祂把我們放在那樣的情境當中。在我們有需要的時候，祂是我們隨時的幫助，因此讓我們來照著祂對我們家庭所存的心意來打造，在這個過程中，也讓我們重新定義家庭到底該怎麼運作才對。你不僅是具有不可思議的超能力，基本上無人抵擋得了你。你將會培養出能夠摧毀仇敵詭計的孩子，那會使你的家和所處世界都看起來更像天堂。

### 🙏 禱告引導

求神賜給你「藍圖」，好讓你知道自己與家人一同要建造的是什麼。與祂一起築夢，想像當你更懂得怎麼培養出被聖靈所引導的孩子時，那將會使一切有何改變？

花點時間來稱頌讚美神，即便你可能感覺自己現在正在面臨爭戰。寫下你從神那裡領受過的應許，並且來一一地默想吧。
