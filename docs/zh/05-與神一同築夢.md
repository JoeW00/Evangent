# 與神一同築夢

我們都從父母承接了正面和負面的東西，他們一樣也承接了上一代傳承給他們的東西。有些人說負面的是累代的咒詛或謊言，不論你用什麼稱呼，我們從雅各和拉班的故事看到（見創世記三十章），我們有能力選擇從上一代承接什麼。

雅各一直以來為岳父拉班工作，但生了約瑟後，他已準備好要帶著家人回本鄉了。拉班不想讓他停止放養自己的群羊，因為他知道雅各使他蒙福。於是，拉班開了一個約瑟無法拒絕的協議，他說：「請為我定你的工資，我就給你。」（創世記三十章28節）。雅各說請給他所有有斑點的羊，剩下的就留給拉班。拉班同意了，但他卻帶走了所有有斑點的羊，心想這樣就能避免雅各從羊群中再繁殖出有斑點的羊，沒想到雅各有辦法扭轉整件事情為自己效力。

雅各拿了一些樹枝，在上面割了一些斑點和條，插在羊喝水的水溝和水槽裡。那些沒有斑點的羊，看著有斑點和條的樹枝喝水，就生出有斑點的羊。這些羊因著改變所專注的，而停止繁殖出上一代的特徵。耶穌是我們的牧者，我們是祂的羊。祂可以透過改變我們的焦點，來改變我們生命所出來的東西。

若我們只是告訴自己不要重蹈覆轍，就很容易複製過去的傷害給下一代。換句話說，若只專注在父母對我們所做的，即便這麼做的用意是想避免犯同樣的錯，卻仍是非常有可能會複製一樣的行為在孩子身上。外在的行為由我們心裡所想像的來決定，擁有內在誓言的孩子，很容易不斷在腦海裡播放他們父母曾經對待他們的方式。其中一個改變的方法就是改變想像中所看到的事物，就像雅各改變他的羊群所看到的一樣。

結果是雅各原始的這批羊所生的後代都一同被影響。若我們不定睛在天父，我們只會複製父母的樣式。如果我們改變所看和所想像的，我們就能改變教養孩子的方式。如此，我們的孩子就會開始承接許多正面和屬神的特質和事物。他們生命就沒有那麼多不健康、需要被打破的循環，因為我們已經替他們打破了。

## 打破循環

有些負面的行為累代地在家族裡不斷上演，甚至有些循環永遠都不會被打破。很多父母告訴我，當他們極有壓力的時候，會突然間變成自己父母的樣子。他們不是故意的，但他們回應和對孩子說話的方式就如同自己的父母。

我不只幫助過許多面對類似問題的父母，我自己也曾是如此。我記得坐在我媽媽家裡的地板上，哭著問她為什麼我一直在複製爸爸的行為，尤其是那些我不喜歡的地方。我的行為沉重到我已經受不了了，我沒有力量去改變。當時，我已經重生得救了，但我覺得自己失去了人生的掌控權。我無法完全地活出神在我生命中的改變，只能複製我父親曾對待我的方式。

耶穌在拿撒勒也遇到類似的事情，祂回到家鄉，在會堂裡教導，使眾人驚奇（見馬太福音十三章）。這出乎他們的意料，因為他們只知道祂是木匠的兒子。他們不知道祂從哪來的智慧和異能。聖經說：「他們就厭棄祂。」（馬太福音十三章57節）但耶穌並沒有說他們厭棄祂，而是說他們不尊敬祂（57節）。

故事的結局是「耶穌因為他們不信，沒有在那裏行很多異能。」（58節）。短短幾節內，我們看到因著厭棄、沒有尊榮及不信的後果，就是整座城無法領受及經歷天父放在耶穌身上的恩膏，因為他們只從祂世上的家庭來認識祂。

在我哭著問我媽為什麼我一直複製爸爸的行為後的幾年，我開始學習尊榮真正的意思，以及它在屬靈層面的影響。比爾．強生牧師說尊榮是：「去為別人的所是而歡慶，並且不要因為他們不是什麼而倒。」2人在生命的某個階段，有可能尚未能完全活出神口中所說的自己，他們有時候在行為上就是沒有與自己裡面的真實身份相稱。我們需要注意，不要被人們的失敗絆倒。當人們沒有活出神給他們的身份時，我們就很容易不尊榮與被絆倒。當我們不尊榮人，我們就看不見神創造他們最原始的樣式。

我意識到我這一生就是不斷專注在我父親的失敗上，我從來沒有看過他活出神創造他的樣式，也因此我從來沒有為他的所是而歡慶，我一直被他的失敗絆倒。有一天在禱告中，突然間神讓我看到我爸爸以前如何飛跨東西兩岸，下飛機與我見面後，就馬上又與我一起上飛機，帶我飛回他的家裡。不論要飛多遠，我爸爸會用他所有的假期來陪伴我。

當我看到畫面，我明白即使他有缺陷，但我爸爸也用這樣的方式彰顯出天父對我的愛。那一瞬間，我開始尊榮我爸爸，過沒多久，我再也沒有重複他的過錯。

可供下載

就屬靈層面而言，尊榮與不尊榮彷彿網路上的下載鍵，按下去後，就會下載該按鍵設定要下載的東西。有些按下去會下載病毒，有些會下載電影或電子書。有些甚至可以清除過去下載的病毒。

每當我們不尊榮父母或被他們冒犯—就像拿撒勒人對耶穌的感受一樣—我們就好像在下載家族裡不屬神的東西。樣地，當我們不被他們曾經的失敗絆倒，就有辦法用神創造他們的眼光來看他們，好像按了新的下載鍵，就能下載屬於神家族裡的東西。若我們重複了上一代的錯誤，可能代表我們還活在不尊榮裡。

既然講到下載，我認為我們需要學習記憶是如何存在我們腦裡的「硬碟」。我最喜歡的一部兒童電影叫做《腦筋急轉彎》（Inside Out）。它的場景主要在12歲女主角萊莉的腦中。這部片有許多值得我們學習的，但令我印象深刻的是編劇是如何呈現我們儲存記憶的方式。萊莉的腦中有五種情緒：快樂、難過、恐懼、憤怒和厭惡。每一樣情緒也都有代表性的顏色，快樂是黃色，難過是藍色，恐懼是紫色，憤怒是紅色，厭惡是綠色。

所有萊莉生命的回憶都會引出一個情緒，然後就會在她腦中儲存到對應顏色的球體內。她第一個與家人有關的回憶，因為帶給她快樂，所以在腦中呈現的方式就是一顆黃色的球體。有些與家人的互動，讓她感到憤怒，所以那樣回憶就是紅色的球體。電影描述很棒的是，我們的回憶其實都會包裝在我們當下所感受的情緒裡。

每當我們讀一本孩童教養的書，我們都會汲取新的資訊。很多時候，我們很期待要應用所學的技巧、工具和教養方法，所以我們就將所有的資訊存在腦中。問題來了，我們學習時的狀態，往往都是興奮和快樂的，所以我們將書中學到的所有資訊包裝在快樂的情緒裡。不只這樣，通常我們也是孩子睡了或是渡假時才有空讀這類的書籍。，我們都是在平靜的情緒狀態下學習和消化這些新知。

問題是，我們往往不是在快樂或平靜的狀態下需要這些教養資訊。往往是我們生氣或煩躁的時候，小孩不願意刷牙或準時上床睡覺的時候。或者已經急著要出門，小孩還沒穿上鞋或忘了為他們準備午餐的時候。換句話說，我們需要這些資訊儲存在生氣、厭惡或煩躁，而不是快樂的情緒裡。

這或許就是許多家長會納悶，為什麼每次趕著孩子出門，當下只記得該情境下早已用過上百遍的教養方法，而這些方法往往都與自己父母所使用的一樣。當家長恢復平靜時，他們驚訝自己又能想起書裡所學的，但就是在他們最需要的時候卻會瞬間記憶斷片。

我不是在說，為了記住書裡的資訊，要用生氣的情緒來讀書。而是，我們可以利用想像力，想像自己在那些特定的情境和情緒下使用這些方法，以至當我們真的生氣時，這些資訊隨手可得，不會難以捉摸。

替換彈匣

我們也可以用「觸發」的觀點來思考，相信你們都知道什麼是觸發點，以及哪些事情會觸發你的情緒。觸發點就是當它碰觸到一些事情時，會引起你一些反應。以準備幫孩子入睡為例，那是許多父母最容易被觸發的時候，我們想要孩子刷牙、上廁所、拿本書，然後躺在床上，我們想要孩子照著我們的話去做。哄睡的階段其實不是真正的觸發點，真正的觸發點是孩子們不聽話或拖延。當孩子沒有馬上聽話時，他們就觸發了父母，而父母就會有所反應。

我們可以用槍的圖像來談論觸發點（譯註：觸發點的英文也等同於手槍上的「板機」），我們心裡的彈匣裝了什麼，孩子不聽話時，就會射出什麼。如果你不熟悉槍的用詞，彈匣是裝子彈用的長方體，安裝在槍體握把的位置，子彈就是透過彈匣送入槍體中。

在彈匣裡裝子彈，不代表彈匣已經扣上槍了。這就是為什麼我們學到關於教養孩童的新工具，總是在最需要的時候過了才想起。好比我們的彈匣裡裝了新子彈，但卻把它擱置一旁，仍是從舊的彈匣發射。當孩子們碰觸到我們的觸發點，就如同是扣了槍的板機，我們不明白為什麼射出去的都是過去的子彈，不是已經裝了新子彈了嗎，為什麼我們對孩子們的反應還是一樣？只因為彈匣裝了新子彈，不代表就有成功上膛。

我們必須明白，情緒就像是彈匣，子彈就像是我們教養孩子的技巧和工具。我可以學會世上所有的工具，但若我只是在平靜的狀態下學習，它們並沒有被裝進我「憤怒」時的「彈匣」。這時孩子觸發我，扣下了板機，沒有新的工具或方法會出來。好消息是，一旦明白了這一點，我們就知道如何將這些工具裝進我們想要的「情緒彈匣」裡。

取代舊彈匣

神差派約書亞帶領以色列人進入應許之地，這是他們40年前就可以進入，卻到現在還沒進入。神對他們說：

只要剛強，大大壯膽，謹守遵行我僕人摩西所吩咐你的一切律法，不可偏離左右，使你無論往哪裏去，都可以順利。這律法書不可離開你的口，總要晝夜思想，好使你謹守遵行這書上所寫的一切話。如此，你的道路就可以亨通，凡事順利。

約書亞記一章7到8節

為了奪下應許之地，約書亞必須剛強壯膽，為了能剛強壯膽，神的話語不可離開他的口，必須晝夜思想。因著晝夜思想，便能在一切事上亨通順利。思想或默想就是深刻並重複地思想某句話、經文或故事。好像一頭牛反覆咀嚼食物，我們也應在思想中反覆「咀嚼」神的話語。

默想能使我們成為凡事順利及亨通的父母，我們能在安靜的時候，默想那些容易使我們被觸發的情境。可以坐在餐桌，喝著咖啡，想像著當晚的哄睡儀式會如何進行。我們可以想像著孩子們以往的行為模式，可能是拒絕刷牙或拖拖拉拉的上床。想像的時候，那些煩躁及不舒服的感覺也會湧上，這個感覺很重要，因為我們要將新學到的技巧和工具存放在這些感覺裡。感覺到這些負面情緒時，接著就可以想像自己用那些新的工具及技巧來回應孩子。

與其發脾氣或對孩子大小聲，我們可以默想自己平靜地告訴孩子們，準備上床時間可以慢慢來。是的，我們確實在默想自己允許他們不聽話和拖延。是的，這或許會促使他們去觸碰你的觸發點及忍耐的極限。聽起來很荒謬，但事實是，不論你有沒有促發他們，他們都有可能會觸發你。但我們要默想自己忍耐地、平靜地告訴他們事情的後果，並讓他們選擇要怎麼做。我們默想決定權仍在自己手上，不是讓孩子的行為，決定我們的反應。

默想的時候，我們也要邀請聖靈在我們的想像中來賜我們能力，接著就繼續過著日常的一天。晚上，孩子們仍然會做他們過去每晚都在做的，但這一次，我們可以選擇不一樣的回應，現在我們有選擇權。而我們感到不耐煩時，我們想像的畫面可以供我們使用。

很多時候，我早上默想完情境後，當天就會遇上。我開始煩躁時，就會想起早上默想的畫面，聖靈也會幫助我用所默想的方式來回應孩子。當我已經看到新的行為模式，就不需要回到從前一貫的作法。因此在孩子的管教上，我不再無能為力。

切記，孩子行為表現良好，不代表我們就是優秀的父母，父母的優秀與否取決於我們的行為和回應方式。如果我們只告訴神，孩子有多難搞，那就好像一直專注在巨人般的挑戰，而不是渴望結岀的果子。如果我們自己在腦海中重複播放所不要的情境，我們就好像沒得選擇的羊，只能看著眼前的畫面和家族過去的行為模式，複製到我們下一代身上，而不是複製我們的牧羊人顯明給我們看的。沒有人能替你做，也沒有人能替你選擇自己腦海裡想要專注在什麼樣的畫面，這是我們必須為自己做的紀律和決定。

### 🙏 禱告引導

請求天父來轉化你的心思意念，並斷開任何從父母繼承的不健康循環。求祂幫助你想像當你被觸發時該如何回應，求神給你智慧去明白如何好好愛孩子，同時又能自制。花點時間來想像神教你的回應方式，而不是你過往習慣的模式，並同時感受那些情境下會有的情緒和感覺。
