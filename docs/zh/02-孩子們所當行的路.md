# 孩子們所當行的路

說到養育孩童，最常被引用的經文就是「教養孩童，使他走當行的道，就是到老他也不偏離。」（箴言二十二篇6節）不幸的是，我們很多人都沒有正確的解讀這節經文，我們是這樣理解的：「教養孩童，使他知道所有不當行的道。」

如果我們把焦點放在所有孩子們不該去做的事，他們就會無路可走，只知道有很多不該走的路。如果我們用對於這節經文的錯誤理解來育兒，我們可能會冒著培養出只會對生活做出各種反應的孩子這樣的風險。他們會更小心地避免做不該做的事，而不是去專注成為自己該成為的人。「使他走當行的道」這句話涵蓋到的是一種方向感，意味著我們的孩子能夠帶著明白自己出生在這個世界上的原因而活。他們能夠知道自己存在於這個世界的意義。

如果用飛機的登機證來做個比喻，為了要讓我能搭上正確的飛機抵達目的地，我的登機證上會寫著我該去報到的那個登機口。登機證會指引我去到那唯一正確的登機口，同時也在告訴我去忽略其他所有的登機口。如果今天登機證上只列出所有我不該去的登機口，我可能會很著急的在機場裡亂竄，但當我一知道自己的登機口後，我就可以準確地朝著正確方向前進。

著急地亂竄可能是很多人的人生縮影，但這並不是我們希望孩子們過上的生活。一旦孩子們學會了「當行的道」，他們就可以開始尋找能使他們抵達目的地的指引。為了使孩子走當行的道，我們也會需要告訴他們不當行的道。參孫是聖經裡關於這個教導很棒的例子，一名神差派的天使臨到他母親，說：

「向來你不懷孕，不生育，如今你必懷孕生一個兒子。所以你當謹慎，清酒濃酒都不可喝，一切不潔之物也不可吃。你必懷孕生一個兒子，不可用剃頭刀剃他的頭，因為這孩子一出胎就歸神作拿細耳人。他必起首拯救以色列人脫離非利士人的手。」

士師記十三章3-5節

天使囑咐參孫的母親清酒濃酒都不可喝，不潔之物不可吃，也不可以剪參孫的頭髮。天使吩咐了他和他母親不能做的事（不當行的道），但在後面清楚了告訴她需要如此行的原因，讓她知道參孫即將成為的人和未來將要完成的事（當行的道）。

想像一下參孫在炎熱的夏日向媽媽哀求想要剪頭髮，我相信參孫的母親需要不下千次地向他重新解釋不能這麼做的原因。

如同所有小孩，我很確定當他的媽媽向他說不的時候，參孫一定都會哭鬧的喊：「為什麼？」但一次又一次地，她都可以用神的話語來回應他的疑惑。

「你是要拯救以色列人脫離非利士人的那名勇士。透過把頭髮留著，你可以變得強而有力，敵人將完全碰不到你的一根汗毛。」

她必須要告訴他不當行的道，但同時也留心於他當行的道。我們都知道接下來故事的走向，不幸地，參孫迷失了方向，也從這之中學到了慘痛的教訓。但如同神的應許，他最終仍然成就了神當初對他的心意。

與神相遇的重要性

關於參孫的故事，我最喜歡的部分是他的父母都分別得到從神而來指引的橋段。他們對兒子的教養方式是基於與神之間特定的對話和經歷而來，而這樣的故事在聖經中也不是第一次發生。在新約，約瑟與天使在夢中相遇，透過這次的相遇，他明白該如何去面對與馬利亞之間的關係，也收到要帶全家人搬家的確切時間和地點。每次只要約瑟需要時，神的糾正、引導和保護就會臨到。相信神當時會為這些父母做的，現今仍會繼續。

從另一個角度來看，我們從聖經裡可能沒辦法找到所有關於教養孩子問題的解答，我們會需要代表我們的孩子去找神，就如同聖經人物所做的一樣。每個孩子都是獨一無二的，他們被神創造在這個世界的目的也是獨一無二的，因此神對每個孩子的培養都有客製化的策略，無法一個模板全數套用。

參孫和撒母耳的父母都因著他們孩子的緣故與神相遇，但收到的是獨特且不相同的引導。耶穌的一生完整地活出了一個對天父順服的生命，因為他的父母也是如此，他們會去做任何神要他們做的事。作為父母，他們會分享關於自己順服神的故事，以及順服帶給他們生命的影響，並教導孩子相同的生活模式。如果我們能夠在教養孩子前，先選擇來到神的面前，我們將可以培育出習慣先去尋求神心意的孩子。

我們的女兒出生時，神告訴我的妻子：「記得不要去控制這個孩子。」這個提醒ㄧ次又一次地幫助我們。我和妻子一直以來都會刻意給孩子們很大的自由，讓他們可以自己做決定，並且學習去經歷這些選擇所帶出的結果，但面對這個女兒，我們必須加倍留心地去做這件事。

某天下午，我們在陽台休憩時，她不知道從哪邊抓來一個很髒的軟塑料，並開始咀嚼。我開始告訴她不要去咬，但很快地，我想起神的提醒。我往後退一步，重新來過，向她解釋如果她想繼續咬的話是可以被接受的，但是塑膠裡面所含的化學物質可能會影響長期的身體健康。然後我給她自由去選擇，由她來決定是不是要繼續這麼做。我六歲大的女兒思考了幾秒鐘的時間，最後決定不再咬了，並且把軟塑料丟進垃圾桶。

一個從神而來的簡短指令，幫助我們能夠帶領女兒自己做出更好的決定。

實際訓練比口頭解釋更重要

在箴言二十二章6節當中，希伯來語「教養」的意思，能夠幫助我們更好地理解這節經文。這個詞的意思包含了「將東西放進嘴裡」然後「讓他去品嚐」1。身為家長，不能只單用話語去告訴孩子什麼是他們該做的，我們必須給予他們實際體驗的機會。這就像是你嘗試去解釋你最喜歡食物的味道，你可以極盡所能地形容，但除非你讓孩子有機會親自吃過一次，他們才有可能明白為什麼你最喜歡吃這個食物，以及他們自己會不會喜歡。

我們家是當我女兒在學校遭遇到痛苦的經歷時，一起體會到了這件事。當時她的年紀已經到了同學會開始彼此說傷人的話的時候，她在遊戲場被欺負。她就在描述某個特定的情境，她告訴我當她感到受傷時，整個人會呈現關機、不知所措的狀態。我開始給她一些建議，教她當下次再遇到相同的事情時可以怎麼樣更好地去回應同學，我教她如何去設立界線，並且嘗試和對方說明他們的行爲將對她造成負面的影響。正當我自我感覺良好，覺得自己清楚地教導她處理這個情形的策略時，我發現我的分享並沒有真正地進到她心裡面。我在教導她下次該怎麼做，卻沒有讓她有機會實際演練。我當時想不到有什麼樣的方式能夠實際讓她體會我所在教導的事情。

我將我的心快速地去對焦於上帝，祂馬上給了我一個點子。我帶她重新走過稍早在學校發生的事，並在要教導她做出不同回應的時間點，請她跟著我將句子覆誦一遍。

正當情景演到使她不知所措的環節時，我說「他把球從你手中搶走，然後對妳說：『妳不能跟我們一起玩！』，然後你要回他...」她馬上就接著我的句子講下去，很好地應對了那個欺負她的小男生。我快速將幾個場景重複演練，她都能迅速用我教她的話語去做答覆，這麼做，能幫助她去預習當下次遇到同樣的事情時該如何回應。如此一來，我就不僅僅是在對她做口頭的解釋了，而是給予實際的訓練。

我們大腦的構造是，當我們愈針對一個情境去重複經歷或做選擇，我們愈會在腦裡建構出一個思考和行為模式的迴路。在我的想像裡，我們腦中的思路就像一輛被地面上的軌道清楚指引的馬車，我們愈是走在這條軌跡上，就愈能知道接下來該怎麼走。

以我女兒的例子來說，當她愈去選擇關機什麼都不做，這種反應的軌跡就會愈深刻的烙印在腦海裡，如此一來，她下一次會選擇關機的可能性就愈高。她愈不知所措，就會更加失去做出其他反應的能力。在她心中判斷能做出的選擇將會愈來愈少，很快地，在我們意識到發生了什麼事之前，她已經掙扎地偏離所當行的路了。

如果我只有堅持用口頭去教導她做出不同的回應，我只是成功地為她指出一條新的路，她還是會需要自己去開發，同一個情境再次發生時，她仍然會掙扎於如何跳出舊有的反應迴路。但當我帶她身歷其境、完整地練習覆誦我教她的話語時，我已經在引領她前往創造那條新的迴路，這樣當她再次面臨相同問題，她將能夠更輕易地選擇走這條新創造出來的路。

允許什麼事進入到我們的生命中

我們非但需要透過自身的示範去邀請孩子進入一個親自與神有面對面關係的生活，我們也會透過自己的生活方式去塑造孩子其他層面的品格 — 其中有些是我們需要避免讓它進入到我們家庭中的。

我曾經造訪過一間教會，教會的牧師有兩個成年的孩子，我對他們兩位都相當尊敬。每次當我有機會與這位牧師交談時，我都會向他尋求這幾年他養育孩子的智慧。在一次對話中，他告訴我他的其中一位孩子是很懂得感恩的小孩，但不知怎麼地，有一天開始，他展現出自負和不感恩的態度。在他決定要做點什麼之前，他選擇先去禱告。當他這麼做的時候，神告訴他，正是他這位做父親的讓不感恩的心態進入到自己的生命中，以致於也流進了兒子的生命。

神不但沒有讓他去糾正兒子不感恩的心，反而讓這位牧師去向兒子道歉；允許了不感恩的心態進入到自己的生命當中。畢竟這位兒子（就像所有的小孩）只是單純地去模仿了父親的行為。這位牧師後來招待他的兒子出遊，並且為著讓不感恩的心進入到他的裡面而道歉，並且詢問兒子是否願意原諒他，他的兒子答應了。從那一刻開始，他又轉變回一個擁有感恩的心的小孩。

司布真（Charles Spurgeon）是一位極具影響力的英國牧師，他說過：「教養孩童，使他走當行的道 — 但要確保自己也走在其中。」2

養育孩子的禱告

關於聖經裡養育孩子的禱告，我最喜歡的就是稍早提到關於參孫的故事（請見士師記十三章）。耶和華的使者造訪參孫的母親，告訴她即便她過去是不孕的，她仍將生出一名男孩，並將明確的飲食以及育兒方針交給她。然而瑪挪亞並沒有像他的妻子一樣得到從神而來的啟示。他的回應是選擇向神祈求：「主啊，求祢再差遣那神人到我們這裡來，好指教我們怎樣待這將要生的孩子。」（士師記十三章8節）。

神垂聽了瑪挪亞的禱告，耶和華的使者再次臨到，這次他得到從神而來的第一手啟示。這成為了我最愛的禱告的原因是，我對瑪挪亞的渴慕感同身受，希望能從神那裡得到關於我的孩子們清楚又明確的方向。我們從經文中可以看到瑪挪亞與神的經歷，為參孫的生命帶來極大的影響力，從我的生命中也能有相同的看見，每次與神的相交，都深深地引導和改變我們一家人的生命。正因如此，我常常引用這節經文禱告。

### 🙏 禱告引導

神是不是曾經有跟你說過關於你任何一個孩子的事情？如果有的話，可以將這些話記錄下來，祂給了你什麼樣的引導和建議？

將一個你和孩子正在面臨的難關交托尋求神，求祂指教你該怎麼做。向神尋求智慧和洞見，明白孩子當行的路和未來會成為一個什麼樣的人。寫下任何從神而來，新得著的詞或者方向。
