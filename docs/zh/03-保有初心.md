# 保有初心

## 我們並不知道自己在做什麼

當我還在伯特利教會擔任兒童主日學牧師的時候，來自世界各地的人們會前來向我們學習如何更好地開展兒童事工。但我通常會用這句話作為特會的開頭：「我們其實並不知道自己在做什麼。」

接著，我會停頓幾秒鐘，讓大家去消化這個聲明。會眾們是從各國各地特別空出他們生命當中寶貴的一星期，大老遠飛來學習如何在自己的兒童事工上面有所進步。儘管如此，我們卻宣稱我們並不知道自己在做什麼。

然後我會接續解釋，我們並不是在說我們對於事工一無所知；相反地，我在向他們揭露我們團隊之所以強大的原因。我們會確保不論我們事工進展得多順利，我們都不會讓自己陷入以為自己已經成為專家的地步。我們很努力地將自己的心態保持在初學者的狀態，需要完完全全倚靠神的幫助和指引。

在這個章節的開頭，我也想要跟你說一樣的話，我們身為父母最大的優勢將源自於保持謙卑和擁有願意學習的心。我這邊在講的當然不是指我們要拒絕在養育孩子的事上繼續成長或變得更有自信，而是我們要時時與了解自己手中工作的那一位保持聯結，我們需要保持在初學者的姿態。我們不想要走到一個地步是，我們自認完全知道接下來該做什麼事，而不再去倚靠天父的幫助。

身為家長，我們一定有的一個共同經驗是，我們以為已經能夠掌握的事情在一瞬間馬上又變了，然後才發現原來我們只看見冰山一角。我們需要持續地刻意領受神的話語，如果在任何時候我們感覺自己不再需要神了，可以自己來；我們就會陷入謊言當中，就像夏娃當時在伊甸園裡一樣。

養育兒子的恐懼

我的第二個孩子，奧古斯特（August），是我的第一個兒子，當他出生時，我經歷了人生中最令人害怕的兩天。他出生的那日，因著生產我們整晚都沒睡，而他的姊姊隔天一早醒來就開始吐。我們當下想著，不論姊姊生的是什麼病只要傳染給弟弟就慘了，我們感到非常不安。睡眠不足再加上對於弟弟會被疾病傳染的恐懼，我對家裡的人開始變得暴躁不安。一整天，我對我的岳母和妻子都很無禮，情況糟到蘿倫希望我先暫時離開這個家，去找神理解一下我的問題到底出在哪裡，她說在我變得比較好相處之前，都先不要回來。

我帶著我的筆記本去到家裡附近的一個公園，坐下來，並從我的觀點寫下這幾天發生的所有事情。我對神寫說，我很害怕我的兒子會去世，但在那一瞬間，祂對我的心說話：「 問題並不是這個。」

祂繼續解釋，當我女兒出生時，我進入了一個未知的領域，但因為我是擅長面對未知的，所以對於接下來的冒險感到興奮。當我不知道接下來要面對的是什麼的時候，我會把它視為一種挑戰，對我而言，女兒的出生就是一場全新的冒險。神向我指出，生兒子就不一樣了，因為對我來說這並不是一個未知的領域。我不僅已經有照顧孩子的經驗了，我還經歷過一個父子的關係。我在一個父母離異的家庭中長大，我所經歷過的父子關係，是父親總是缺席在兒子的人生裡。

神向我顯明，我很期待的要去當一個女兒的爸爸，卻很害怕要成為一個兒子的父親。祂的話對我而言非常真實，我對神說，祢說對了（祂當然是對的) — 我很害怕，我完全不知道自己在做什麼。祂回答我：「很好，這就是我希望你待的地方。」那些很清楚知道自己在做什麼的人，最終往往不再認為自己需要神。

那天晚上我回到家後，我向每一個人道歉，包括對我的兒子。一天大的嬰孩不會明白道歉是什麼，但我不是在對他的理智、而是在對他的心說話。我告訴他我很害怕，因為我從來沒看過一位父親怎麼樣在家庭和信仰上有好的帶領。我向他承認，我不知道我自己在做什麼，我告訴他我會向其他父親學習，最重要的是，我會盡我所能地向天父學習。有了祂的幫助，我不再感到害怕，我最後告訴我的兒子，我一定會陪伴他成長。從那之後，我再也沒有感到害怕過，我也開始愛上擁有一個兒子的感覺。

身為家長的我們都曾經歷過那樣的時刻，我意識到我不知道自己在做什麼。通常我們會上網或看書尋求幫助，書籍和來自其他家長的經驗談多多少少都一定能給予我們幫助，但他們不是從真理出發的最佳解答。當我們在向同儕尋求智慧的時候，我們像是在求助於「承包商」，但當我們向神尋求，我們能夠直接得到原創「建築師」的解答。祂是唯一真正了解我們孩子，並且知道他們需要什麼的那一位。

為男孩買飛刀

我的岳父有三個女兒，她們現在都結婚了，所以他有三個女婿。因為他的女兒們都不喜歡打獵，他最喜歡一起跟我們做的事情之一就是帶我們去打獵。有一年我們要去新墨西哥州進行為期一週的麋鹿狩獵之旅，有佔地一萬三千英畝的野地讓我們去捕獵麋鹿。對我們這些年輕人來說這是頭一次，我們以前有一起獵過鴨子，但跟獵麋鹿比起來，根本不是同一個等級。

我將一切準備妥當，還去上了好幾個小時的狩獵安全課程，準備好出發 — 或者我當時是天真地這麼認為的。當時的計畫是，我會自己去新墨西哥州和他們會合，因為在那之前我還有其他行程要完成。就在要出發的前幾天，我發現我還沒拿到新墨西哥州的打獵資格證，我嘗試了各式各樣的方法想在抵達新墨西哥州前的最後一刻拿到，但一無所獲。我知道對很多人而言，這不是什麼特別大的事情，但我很在乎。很快地，我的心情盪到谷底。

我開始質疑自己真的想去打獵嗎，我質疑自己的男子氣概，並開始問自己像是這樣的問題：「我的潛意識是不是不想去，所以才毀了這次的行程，因為我壓根不是這樣的男人？」（我的意思不是會打獵和殺動物才是真男人，但在我當時的人生階段裡，我期望能透過這一趟狩獵之旅重新建立與男人這個身份的連結）。

我感覺神邀請我去和祂聊聊，我闔上眼，想像與天父在一起的畫面。很快地，祂讓我想起當我還小的時候，我用我的零用錢買了些飛刀，還因此闖了很多禍，因為我想拿它們去投擲所有的東西。天父讓我想起我是一位喜歡打獵的男生，但在成長中的某個階段裡我跟這部分的自己失去連結。祂向我顯現這一趟旅程將會喚醒一直存在於我內心裡的東西，也要我別去擔心證照的問題。

在我要睜開眼睛之前，神給了我一個關於我兒子的畫面，然後告訴我要買飛刀給他當作聖誕節禮物。他當時才五歲，但神確確實實地叫我買飛刀給五歲大的兒子。不僅我不知道要怎麼帶我的兒子，我也開始懷疑神是不是真的知道？

當我抵達新墨西哥州時，我的岳父已經為我準備好打獵的資格證了，我竟然讓自己為不必要的事情大費周章又壓力山大。接著我上購物網站，替兒子買了十二把飛刀和一個很不錯的保護套。

當為期一週的狩獵之旅快結束時，我們各自帶著打到的鹿肉回家。在野外獵麋鹿是一個不可思議的體驗，˙那喚醒了我內心的某個部分。我比太陽還早起很多、準備好所有用具、耐心地等候、在壓力下保持冷靜，並且還提供家人們一年所需的營養。我不是只是去超級市場買肉，這是我跋涉在一萬三千英畝的野地才收獲到的戰利品，原來我是這樣的男人 — 直到那刻我才真正意識到。

打獵的過程也深深觸動我的心靈。我選擇去奪取麋鹿的生命，讓牠可以滋養和支撐我的整個家庭和很多的朋友，這跟我的選擇是如何奪取了神羔羊的性命是相似的，祂如今養育了我的生命。我對於麋鹿犧牲牠的生命去養育他人的感激，之於我對耶穌的感激是相似的（但當然還是遠遠不及）。

幾個月後，我們家一起慶祝聖誕節，我等不及要送禮物給兒子了。平安夜裡，我帶他出去和我一起砍些木頭回來，他問我為什麼要這樣做，我告訴他，等到明天你就會知道了，因為這也是禮物的一部分。

到了隔天早上，他先打開了其他所有的禮物，最後才是飛刀。當他打開保護套發現禮物的內容物的時候，他跳向我，給了我一個擁抱，這個擁抱有別於過去所有他給過我的擁抱。我給兒子的這份禮物，不僅僅是為了要讓他在接下來的幾年裡可以去使用，更是成為一名年輕男子的象徵。從過去所有我和他討論到關於飛刀的對話中，我兒子明白，他使用飛刀的自由是取決於他證明自己能夠被信任賦予這樣的工具。當他看到飛刀，他就知道自己已經得到我的信任。

飛刀的意義遠大於一個很酷的玩具，它是一種認可，代表他已經能夠去掌握危險的東西並且保持自身和他人的安全。當他在拆保護套看到飛刀的時候，就好像也看進了自己內心真正的一部分。飛刀對我們父子倆都意義非凡，我很享受那個時刻和那個擁抱，但更觸動我的是，神完全知道怎麼幫助我去帶我的兒子，五年前我還很害怕成為他的父親，但有了神助攻後，我現在看起來像是知道自己在做什麼了。

## 建立信任

我知道給一個五歲大的男孩飛刀聽起來很瘋狂，但他現在七歲了，並且擁有十八把飛刀、幾把摺疊刀、三把斧頭和一把彎刀，總共超過四十把刀。折疊刀除外，他每一個都能夠投擲到我們一起做的標靶上。

當他三歲拿到第一把刀時，我花時間向他解釋，他能夠擁有和使用刀具的自由將取決於他與這些刀具建立了多少信任，如果當我在他身旁的時候，他都能夠正確的使用刀具，那我對他的信任就會增加，下一次就可以賦予他更多自由。只有一次他破壞了一點信任，不得不失去一些特權，即便在那種情形之下，當我給他重建這份信任的機會時，他也有做到。

他幾乎每天都拜託我給他使用他的迷你版瑞士刀，我們只要一有機會，就會一起坐下來削木頭。我會握住他的手，一起將木頭往遠離身體的方向削，我示範給他看刀片是鎖不住的，所以就算收起來了還是很容易會被傷到。​​透過這種沒有鎖定機制的摺疊刀（最危險的種類之一）去學習保護自己的安全是很有幫助的，他知道自己必須特別小心，否則刀片會割傷他的手。

一次又一次地，他保持小心和專注，建立了更多我對於他的信任，我也確保讓他知道我信任他。有很多次我必須拒絕他的邀請，因為在當下我沒有時間陪他一起坐下來使用刀具，在這種情況下，我也會跟他討論他回應拒絕的語氣、聲調和態度，也決定了我能夠信任他使用刀具的成熟度到哪。如果他無法接受我向他說不，我也沒有自信他能夠在我說好的時候能將刀具好好做運用。

有一次，他拿著飛刀離開標靶區，在走回投擲線的路上不小心被灑水器絆倒了，他將刀子握住、手伸出去，用身體著地，完全沒有用手去支撐。我看到後趕緊將他扶起，他也開始大哭，我扶著他，並且與他討論剛剛所發生的事。

我很擔心他會傷到自己，但這並不是他在乎的。他滿腦子擔心的是他跌倒了會破壞我對他的信任，我向他確保這件事並沒有讓我對他的信任有絲毫的減少，會跌倒純粹是一個意外，我也向他申明他處理跌到的方式，反而增加了我對他的信任。他證明了即使在那樣的情境下，他都能掌握好刀具。我擦乾了他的眼淚，然後帶他繼續投擲。

我平時將刀具們放在他拿不到的地方。但有一天他學會了如何在沒有我的幫助下取得。我看見他拿著其中一把刀時，我溫柔地將它從他的手中拿走，並告訴他，當他選擇在沒有我的允許下拿取刀具時，就已經破壞了他長期與我建立的信任，在接下來的一個多禮拜，他都不被允許拿刀具。

他非常地沮喪，我在他哭的時候擁抱著他。我告訴他，即使他破壞了信任，我還是會給他機會去重建，並且詢問這是不是他想要的，他當然願意。那週結束後，我拿出他的刀具們並邀請他一起去砍些木頭，正當我們準備要開始進行活動時，我提醒他，接下來他對於刀具的使用將決定我會更信任他，還是更多地破壞這份信任。這次，他比以往都還要謹慎，不僅修復了我對他的信任，還建立了更多。

這些刀具成為非常實用的教材，教會了奧古斯特生命中最寶貴的幾堂課。當我們談論到朋友、車子或者槍枝時，我都會拿刀具和信任程度來做比喻。如果他想要擁有開車的自由，他知道必須先建立信任，如果他想成為一位好朋友，他也明白建立信任的重要性。

他從使用刀具中學到的的道理，影響了他生活中其他許許多多的領域。這個道理使他成為我認識的人當中，最值得信任的一位（他現在也才七歲而已）。我只要向他說出「信任程度」這個詞，他就會挺身而出，展現他所能做的。我要跟你分享的是，我花在這些刀具上的錢帶給我們家的影響是無價的，天父不但知道如何在我毫無頭緒的時候幫助我，更是在我還在處理自己身份認同的危機時幫助我。

我每年會幫忙帶領一個父親和兒子的營會，他在他六歲時帶了一把刀具去。我無意間聽到有位比奧古斯特還要大的孩子，問自己的爸爸為什麼不能像奧古斯特一樣擁有自己的刀，當奧古斯特聽到的時候，他用我見過他瞪最大的眼睛看向我，他意識到被信任是多麼地重要，也看見自己在這件事上累積了多少的分數，我對他眨了眨眼，向他點頭示意。

神不只用飛刀來提醒我的身份，祂也透過飛刀幫助我去建立奧古斯特成為他將成為的男人。透過這些刀具，我們父子倆都因著按照神的方法做事，而學到建立信任是多麼地有價值。

向父親尋求

我有一位朋友的內在醫治事工讓人感到不可思議，她會與形形色色從世界各地而來的人見面，從貧民到大明星都有。她的恩賜能讓她看見一個人過去的光景，並且看見他們生命中最根源的問題。因著她的恩賜，很多人開始對她變得依賴，她很努力地引導大家學習去把問題帶到神面前，讓神直接對他們說話，而非透過她。

很多人打給她去詢問神對他們個人的心意，這讓她感到很挫敗，以致於還直接將「去尋求神」（ask the Lord）編輯進她的信箱帳號當中。她經常告訴人們，如果他們拿尋求從人而來的意見的時間去尋求神，或許他們就沒有那麼多問題要問了。透過這些話，她嘗試想顯明的是，我們有多常傾向於向他人求助，而不是來找神。這與比爾・強生牧師的做法相似，當人們期望從他這裡得到建議時，他幾乎不會提供，通常他會回問對方：「你覺得神在對你說什麼？」

在約翰福音十六章，耶穌教導了關於禱告的重要性。祂重新定義門徒們應該提出的問題和請求。「你們什麼也就不問我了。我實實在在的告訴你們，你們若向父求什麼，祂必因我的名賜給你們。」（約翰福音十六章23節）耶穌的意思並不是說在禱告時你不能與祂交談，而是與天父連結的這個管道已經賜給我們了。那位教導耶穌一切的天父也將給予我們教導（請見約翰福音八章28節），在這節經文當中，耶穌總結了祂這一生所作所為的焦點。祂將門徒們引導到天父那裏，使他們擁有和祂與天父一樣的關係；這也是我們身為家長能提供給孩子們的指引，直接連結於我們的「大家長」。

這對我們以及我們對待孩子的行為會產生什麼樣的影響和意義呢？當我們向他們展示我們與天父關係的同時，也在鼓勵他們親自與天父建立關係。當人生走到了某個階段，一定會有需要擁有自己與神之間關係的時候，而不僅僅是一個依附於我們的關係。

這樣使我們得以擺脫初學者爸媽的身份，我們不用對每件事都有解答，我可以將孩子帶到神面前。我告訴他們，當他們遇到問題時，除了詢問我以外也可以詢問神。我們會問他們像這樣的問題：「你覺得神在這件事上，在對你說什麼？」如果他們不知道答案或者還沒有嘗試去問，我就會帶領他們一起去找神。我們每一次的引導，都在訓練他們直接向神尋求幫助的能力，如同耶穌引導門徒到天父面前一樣。同樣的，如果我平常活出的是一種尋求神的生活方式，那麼就能更輕易地引導孩子進入到相同的生活方式當中。

住在海灘別墅一個月

幾年前我在華盛頓哥倫比亞特區的一間教會講道，神給了其中一位牧師關於我的畫面，他詢問我能不能跟我說。我回答好啊，當然，因為我一直希望能夠更多知道神想對我顯明或者是要向我說的話，特別是如果祂還向別人展示了。

牧師開始說，他看到我們一家人在海灘別墅度過三十天的畫面。我聽到後非常興奮，但也馬上意識到除非神行很大的神蹟，不然連為期一週的海灘別墅度假都不太可能發生。牧師繼續說道，海灘別墅的門口寫著「不提供無線網路」。

我聽見神輕聲對我說，如果我能夠連續三十天不使用無線網路，我的家就會感覺像是一個海灘別墅。我當下很快就決定了，這趟旅程一回去，我不會再在與我家人共度的時光裡使用無線網路。當我傍晚開車快到家時，我先將手機拿出來關閉訊號，並開始稱呼我手機裡的飛航模式為「家庭模式」。

我很快就注意到了一些事情，我發現每當我遇到不知道接下來該怎麼做的情況時，我就會去拿手機來避免卡在這樣的情緒裡，我甚至意識到自己只要有一點點不舒服的感覺，我的手就會下意識地想去拿手機（一個成癮的象徵）。沒有了網路，我不再能這樣子做，我不再透過玩手機去逃避現況，我開始轉向神，讓祂來介入，陪我一起解決問題。我向祂尋求幫助，而不是讓手機去使我分心。

在這段期間裡，我看見一個畫面，一個孩子坐在餐桌前，而父母都在使用手機。標題寫著：「我真希望能變成他們手中的手機，這樣他們就會整天握著我，好好的看看我。」這更加激勵我更多與神親近，也把更多的注意力放在孩子身上。

我體驗到神是我們患難中隨時的幫助（詩篇四十六篇1節），我與神的關係和連結並不只是一個理論，而是我的現實。當我更專注於當下而不是偶爾活在當下的時候，我開始可以更多與妻子和孩子們分享神的屬性。我開始更在乎於我是否在我的故事裡扮演一個積極的參與者，而不是去到社群媒體上發布限時動態，很難承認這一點，但我的孩子們再也不需要透過多次的呼喚才能引起我的注意了。

我聽見神問我，如果你的家庭成為一個實境節目的主題，錄影機會記錄發生的一切事情，你會想要怎麼生活？ 這個問題幫助我意識到我家有很多小臺的攝影機（孩子們的眼睛），它們不僅紀錄了一切，還會將沒有剪輯過的片段回放給我。

自從我不再使用社群媒體後，我不再需要透過別人的意見讓我去明白神的心意，光是這件事，就讓我這一次的三十天旅程值回票價。神給我的啟示也不再需要和每天接受到的大量資訊打架才能順利的被我接收。

即使我們住的地方離海灘一點也不近，我開始更多發覺我所居住的城市是多麼美麗，而每一天晚上，我都能夠將自己從工作中抽離，就形同是住進了一棟真正的海灘別墅。

這件事真的發生了。我度過了一個每天都有休息片刻的三十天假期，而我所做的僅僅是關閉網路。透過這個舉動，我能夠再次回到一個初學者的身份。

初學者

保持一個初學者的心態，意味著我們永遠不想離開單純倚靠天父的那顆心。這不是放棄並接受「我不知道自己在做什麼」的心態；相反地，我們去找祂是因為祂非常清楚自己在做什麼。這避免我們進入一個養育小孩的景況是，只依照育兒守則、最新的育兒攻略或者專家的說法去做，這些建議肯定都是有幫助的，有些狀況也確實就會誘發於特定的年齡階段，但與此同時別忘了，每個孩子都是獨一無二的，只有神明白每一個孩子的需要以及該如何供應。

每次當我們詢問神的想法和意見時，我們就是在相信祂的帶領是超越我們自己對事情的理解（請見箴言三章5-6節），同時也在孩子的心中建立相同的信仰。當我們走投無路時，祂總是有辦法，而我們如同耶穌一樣擁有這個與祂連結的管道，我們為什麼還要局限於自己的理解和專業知識呢？
