# 引言

我開始兒童事工的前四年是在紐約，那時平均兩百個孩童來到我們戶外的主日學。我們後拿塑膠布鋪在道上讓孩子們能夠席地而坐，接著我們會拿一疊傳單在校門口等傳單上印有我們的標誌，附上主日學後會發給小孩們的糖果。

我們的主日學是風雨無阻有時氣溫超過華氏九十度，又濕又熱，有時候則是來到零度以下。販不畏天氣，，我們會上前去傳福音。

每當學校鐘聲響起，不用幾秒鐘的時間孩子們就會將走道擠滿，我們則是興奮地去發傳單，並且向那些路過但還沒有拿傳單的孩子，大聲宣傳活動。另一批人則是在卡車上放起音樂，用大聲公去邀請其他孩子一起加入我們的活動。當人潮後，發傳單的人會回來，然後我們準備開始第一堂的主日。

天氣炎熱時，我們會玩水，讓大家保持涼爽。天氣寒冷時，我們的手幾乎直接被凍在麥克風上，這時我們會提供家長和小孩熱巧克力或者熱騰騰的杯麵。大部分的時候，孩子們會坐在鋪了塑膠帆布的地上，家長則是坐在後面的長凳上。我們分享信息的同時，毒品交易會在周圍發生有時候甚至是孩子們的家長，他們趁著不用照顧小孩的時間去賺點錢。

我們一週會服事五天，通常一天會有兩到三場的服事。場一結束，我們就會立馬開車到另一個社區去服務。我們總共有十七個團隊同時在進行這項計畫週六會帶去教會的三千名孩童，我們平均一週會服事到兩萬名孩童。很多時候，我們只有在天黑到無法繼續時才結束活動。

這是一段非常大有能力且滿足的服事生涯，我卻感覺到必須做點改變。我離開紐約來到加州雷丁。在幾年前，我經歷到神的拯救並且從毒癮中得釋放，所以我清楚明白神會做超乎想像的事 — 只是我不常看見罷了。我感覺自己必須去到能夠經常看見神大能運行的地方，並且學習怎麼在自己的生活當中去實踐。

接下來的十年我在雷丁一邊當傳道一邊學習，​​我為生活在拖車屋和低收入住宅區的兒童創建了主日學計劃。即使規模比我在紐約參與的小很多，但目標是一致的：盡可能地去觸及到更多孩童。我將我從紐約學習到的經驗與我在雷丁的所學做結合，在這樣的基礎上，我們看到孩子們開始以大多數成人夢寐以求的方式與上帝同行。

我們看見孩子們為一名無家可歸的街友禱告膝蓋醫治，沒過多久，他已經能在籃球場上做衝刺。我們看見孩子們為他們的朋友禱告，朋友手上長滿了疣，卻在幾日之內完全消失。有另一位女孩因著肩膀扭傷導致無法將手舉起，在敬拜時她聽見神要她舉起手臂並且隨著歌曲的起伏擺動，在她跟神來回爭執了很久，並且告訴神她就是無法將手抬起後，她決定嘗試了。那些扭傷的部位完全痊癒。的父母跟我們說，回到家後，她甚至還能夠做伏地身。

一個每天飽受多次癲癇發作折磨的男孩，得到立即的醫治。這是當孩子們圍繞著他，奉耶穌的名命令癲癇離開時所發生的。在這些年裡我們也見證了無數個孩子被聖靈充滿，常常需要被人協助扶出會場，因為神的同在實在太強烈這樣的故事說也說不完。

我開始去不同的國家和城市我們從神和孩子們身上所學習到的。有一次去到英國，我和另一位講成了朋友，在其中一次聚餐當中她牧師會做但她，就是孩子們被聖靈充滿。她引起了我的注意，因為我分享，就是神在我們教會孩子她不做。當我問她為什麼時，她告訴我她剝奪父母參與那些美好經歷的機會。我被這樣的想法震懾到，並且那一次對話之後，不停思考這個問題。

不久後，我被邀請至奧地利去帶領一場家長和孩子一起參與的特會。前，我都花時間專注於訓練兒童事工的同工，我從來沒有過任何讓家長和孩子一起參與的活動，因而感到緊張。那天的活動，我按著過去訓練同工的方式去規劃，不過我並沒有讓同工們去，而是讓家長們服事。

我永遠不會忘記那四天所發生的事，兒童這麼多年，從未見過如此大能而有力的。父母和孩子們一起深刻經歷到神。其中一個場次，孩子和家長們寫下自己的罪，以及任何因著罪而讓他們去錯誤定義自己的標籤和身份名稱，接著將紙帶上舞台，那裡擺放了十字架，將紙和上面所寫的ㄧ切放在十字架前。

很快的，一位年幼的男孩被阿嬤帶來哭著走向我，透過翻譯員，告訴我他想要將生命交給耶穌，我向他解釋，當他將罪和錯誤的身份定義帶到十字架前，他就已經將生命交給耶穌了。我們花了一些時間為他禱告，神也在當下將祂的生命和愛傾倒在他身上。

在另一個關於如何的課程中，一位女孩在其他孩子們都去拿點心的時候，從頭到尾都在地板上哭泣，我問她發生了什麼事，她告訴我她還無法去做下一件事情，因為她親眼看見並且聽見耶穌。這些家庭所的淚水喜，深深使我內心。

飛回家的路上，我腦海閃過一個對於幾乎像是謊言的想法，我不確定自己是否能夠繼續相信兒童事工體了。當我克服對於這個念頭的衝擊後，我開始思考是為什麼會有這樣的想法。在多年的事奉當中，我第一次經歷到一些重要的真理。其中一個是神的工作首先應該在家中被實現，家才是孩子們應該經常遇見耶穌的地方另一個則是家長應該成為孩子們的第一個牧者，長期以來我們花很多精力培訓領袖，卻沒有充分去教導和訓練到最重要的群領袖。

我花了好多年透過熱情和使命去溝通，改變眾教會對於兒童事工的想法 ，不僅僅是去當保姆，而是要教導和訓練孩子。我飛到各地教導和這樣的理念  — 而現在輪到我需要去轉換了。

神持續在向我展示關於這個啟示的一切，有一天當我在運動用品店看弓和弓箭的用時，一句經文浮現在我的腦海：「兒女是耶和華所賜的產業，所懷的胎是所給的賞賜。少年時所生的兒女，好像勇士手中的箭。箭袋充滿的人便為有福。」（詩篇 127:3-5），神說想要，我可以拿起一袋弓箭去櫃檯結帳，它們既筆直又鋒利，只要花五十美元，我就能帶回家了。

神幫助我認知到，我一直將教會視為一家運動用品店，而兒童事工就像是賣弓箭的區。我一直認為如果爸媽將孩子送來，並且十一奉獻，那麼剩下的就是兒童事工團隊的責任去這些「箭」（孩子），使們能夠筆直飛起，並且命中紅心我相信很多人也是如此看待兒童事工的。

當然，在詩篇 127 篇被寫下的時候並沒有運動用品商店，當時的士們不能跑進商店買箭，自己。士們在這個過程當中扮演相當重要的角色，不僅要深入了解箭，還要將塑造成它們應該成為的武器。

同樣，家長們不能把孩子從教會接出來以後，就期待他們能夠筆直飛起，命中紅心。父母在自己兒女的屬靈建造上扮演至關重要的角色。身為父母，我們在孩子屬靈生活的建造上，比任何人都還有影響力。我們是拉開弓箭瞄準，使他們飛向命定的那位。

我仍然認同主日學和基督教學校，但我不認同家長們應該將教導的責任和孩子屬靈生建造交在別人手中。如同古代的士會和技藝良好的師傅合作製作最精良的箭，家長們也應該和教會及學校合作，去學習如何打磨他們自己的箭。教會可以提供父母銳利的箭頭，學校可以附上他們所需的箭羽，但箭本身的形塑卻是士的工作。

在古代這是生與死的區別，現在也是。我們不能只是送小孩到教會和學校，期待他們透過這些教導就能夠阻止仇敵的詭計，。

從這三個經歷當中，我服事的核心理念轉變了。也是透過這些經驗，這本書得以誕生。作為父母，我們在屬靈的事上擁有一個遠程的武器。當我們學習如何去教導孩子，我們就能夠與孩子一起去塑造他們成為神創造他們成為的人，並且對於黑暗勢力造成破壞。一位好的射箭手能夠去改變戰局的走向，這也是本書的目的：幫助你去這場與世界屬靈爭戰的方向。

一本書無法去涵蓋我這十六年來全職從事兒童，所學到的知識和經驗。然而這本書所分享的已經足夠作為一個跳板，你和孩子更深進神的同在和大能中。
