# 序

*比爾．強生*

以色列人有被教導，應該要常常想著到底該怎麼讓神可以持續超自然地介入在他們的生活中。為了幫助自己不忘記這點，他們在日常談話當中會不斷地提及神過去行了哪些神蹟，也會一天到晚對孩子們講述這些故事。在這麼做的同時，就把一拖拉庫的回憶無縫接軌地傳承下去了。這些一次又一次重複發生的見證會在孩子的心裡打好基底，並形成一套的價值觀，好叫即便是下一代的人也還是能夠不光是意識到神的同在，而且還了解祂的屬性。將來的世代也能夠傳承自父母而來的產業，包含與這位應許的神建立良好關係、對祂的良善充滿期盼，並以這些為個人生命的根基。如此一來，以色列百姓便能夠持續照著神的心意居住在神所賜的應許之地裡。

只要做好管家就必會看見倍增—這是適用於生命各個不同領域的屬天原則。神選擇將自己賜給以色列百姓作為產業，同樣地祂也把自己獻給我們，不過做父母的卻有責任要訓練好自己的孩子，好叫他們能夠知道並管理好這份恩賜。我們或許可以也很常把自己應該要把孩子訓練好的這份責任委託給執得信任的老師、教練或牧者，不過引導孩子與神同行的這個角色還是最先落在我們的頭上。不管我這一生選擇要怎麼過，我最先該牧養的都不是別人，而是我的家人。

《孩子的屬靈覺醒》這本書就是以此為前提，神有給每位身為父母、祖父母或是照顧孩童者一份專屬於他們的恩賜，好讓他們可以去挑旺孩子對神的渴慕。詩篇七十八篇講到一整個世代都要曉得自己的身份是：「好叫他們仰望神」（詩篇七十八篇7節），這也正是本書努力要達到的目標。

過去近十年多來，賽斯．達爾一直是伯特利牧養團隊中不可或缺的一部分，直到如今他都還在持續地影響著我們。當他在我們團隊裡的那段期間，孩子與家庭都可以很自由地去探索如何與聖靈同行和超自然大能。我們深信聖靈沒有大小之分，因此如果遇到需要醫治的癌末患者，我們很常會讓孩子來為這名患者禱告。如果有個領袖團隊很渴望聽到神對他們說話，通常也一定會有小學生一起來服事他們。當神榮耀的雲彩在伯特利教會主日崇拜裡彰顯時，所有的大人都不知所措，畢竟面對到這樣的經歷很少有說明書告訴我們該怎麼做！不過孩子們卻很懂，他們馬上毫不猶豫地張開雙手奔向神同在的榮耀彰顯，在那裡跳舞與歡笑。一旦孩子被成全可以一無所限地去探索與神的關係時，跟上他們的腳步就算為是我們這些大人的智慧了。

《孩子的屬靈覺醒》裡面有許多實用的工具、符合聖經原則的亮光，以及第一手的家庭見證，這些家庭都被賦予能力去抓住神為他們孩子所預備的一切。除了有過去多年來服事成千上萬個家庭的豐富經歷，賽斯透過自己成為父親這個角色也習得了不少經驗，這使得他的分享充滿智慧。神國要在這地上完成的使命中，孩子們扮演著不可或缺的角色，他們是「勇士手中的箭」（詩篇一百二十七篇4節），就好像賽斯所寫的：「沒有哪個勇士可以拿著不是由自己磨亮的兵器就上戰場打仗。」

我鼓勵所有會參與到教養兒童的人都來讀這本書，讓它來鼓勵並挑戰你去引導自己孩子在屬靈方面的訓練，讓他們打從心底開始認出神的同在，直到我們都能夠安居在神所賜的應許之地裡—也就是使在地能夠如在天一樣。

比爾．強生
美國加州雷丁市伯特利教會
著有《我家也有巨人終結者》和《天國生活》
