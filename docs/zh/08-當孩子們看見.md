# 當孩子們看見

我早已算不清楚曾經被問過多少次，如果孩子們看到一些屬靈的事物，到底該怎麼做才好。「我的孩子晚上會看到些東西」是我很常從父母那裡聽到的說法，大部分如果孩子在房間裡、床底下或是窗戶外看到一些東西，父母就會提出這樣的疑問。而大部分的家長會想要聊這件事，就是因為他們希望知道該怎麼做好。雖然知道該做什麼固然很重要沒錯，但我們同樣也需要知道有哪些事不該做。

什麼不該做

有位家長聯絡我，想要談談她的兒子。他晚上會看到一些東西已經有一段時間了，不過最近他會看到有很大的翅膀蓋在他房間的窗戶上。從我們的談話中間，不光是可以感覺到她很擔心—也可以感受到她有被嚇到。我請她盡可能地告訴我關於兒子這個經歷的所有細節，她有問過他關於翅膀的事，以及那對翅膀看起來的樣子（詢問孩子更多細節永遠都是個明智之舉）。他說那對翅膀是白色的，只要翅膀蓋在他的窗戶上，那些黑鳥就飛不進去他房間了。在她更多向我說明他所看到的細節時，從她的言談之中我聽到更多她內心的恐懼。

不過對我而言，反倒是她講得更多，我就變得更加地興奮不已。

「這太棒了，」我說，這完全出乎她意料之外，但她是一直等到我藉由「那些黑鳥飛得進去嗎？」這個問題重新引導她去注意到這個事實，她才搞懂我說的太棒了到底是棒在哪。

「飛不進去，」她回答我：「天使們不讓牠們進去。」

「所以你兒子在睡覺的時候，有大天使在保護他，所以魔鬼都進不去嗎？」當她聽懂了我所描述的情形時，她的恐懼就離開了。

不論是她或是她的兒子都沒有注意到神的保守，因為他們都只注意到仇敵嘗試要進到他們家裡，即便那些「黑鳥」根本進不來，但是恐懼卻已經潛入，也正是因為這份恐懼使得仇敵還沒有放棄嘗試。

但當這名母親意識到神正在做的事時，她便重新聚焦並且把注意力放在對的地方，於是她不再感到害怕，也開始能夠明白原來兒子看得見這些東西其實是個從神而來的美好大能恩賜。但只要恐懼在作祟，這些恩賜反倒感覺像是咒詛似的，因為它不僅讓這個孩子和他的母親無法好好入睡，還改變了他們的禱告生活到需要尋求協助的地步。專注在仇敵身上的時間哪怕只有一秒鐘，都會使得家中的氛圍有著極大的轉變。而他們之所以會如此掙扎，不是因為那個男孩看得見東西，而是他們在他看到那些東西之後究竟如何去詮釋。

如果孩子會看見一些很可怕或是屬魔鬼的東西，我們必須記住仇敵的首要目標永遠都是要讓我們害怕。為什麼？因為恐懼就形同是在認同仇敵，並且讓邪靈有權利繼續登門拜訪。一旦恐懼進入我們內心，就形同是開了一扇門讓仇敵可以來達到牠的第二個目的：掌控。所以在剛剛這個情況底下，這位母親的兒子看得到些既可怕又邪惡的黑鳥，而是他們母子倆都陷入了恐懼。看到魔鬼後所帶來的恐懼，遠比孩子看到魔鬼的問題還要來得嚴重。

確保門有關上

恐懼就像是在靈界裡一道敞開的門，能讓仇敵肆意地穿梭在其中。如果我們不懼怕，就等同是把那扇門向牠們關上。不過我們也必須銘記在心，魔鬼並不在意靈界裡合不合法的問題。牠說穿了就是個騙子、小偷，這表示牠天性就是會去從事那些非法行為。就算恐懼的這扇門明明沒有打開，但仇敵卻還是有可能會現身。牠們會不斷嘗試潛入，一直試到我們自己為牠們開門為止。

想像一下半夜你的孩子要你去到他/她房間裡，並告訴你說牆外有三個很奇怪的生物貼在那裡。這不代表你應該要立刻想說是不是自己或孩子讓恐懼的這扇門打開了，這可能只是仇敵正在嘗試要你們去為牠開門。父母親很常會在這個時候就開始擔心起來，害怕自己是不是讓家裡門戶大開，殊不知是隨著那份信念而來的恐懼才讓那道門被開啟。

如果你知道自己的門是關上的，你就可以想像仇敵魔鬼都在門外（就算牠們真的在你孩子的房間裡也一樣）猛烈地敲打著門，不斷地對你們大吼大叫，想要嚇唬你們一家人。只要你不幫牠們開門，牠們可能還是會很努力地想要進來。要是牠們真的回來了，只要你不向恐懼屈服，牠們終究會拿你沒轍，也就會摸摸鼻子的離開。

我們希望孩子們看見

首先要避免的就是陷入恐懼，其次身為父母的我們不該做的，就是裝得好像沒有屬靈爭戰這回事一樣。有太多的父母會告訴孩子們說，他們看到的東西不是真的或那不過是他們自己想像出來的。當然了，有的時候孩子們可能會想像出些什麼來，不過我們每次在了解狀況的時候，都不應該先從這個角度去揣測。要是我們不把孩子所經歷到的當一回事，他們也就會不認為自己值得信賴，或是不相信自己所看到的是真的。只因為我們看不見，不代表他們所看見的東西不是真的，只不過是我們看不到罷了。

身為成人的我們看不見靈界，通常是因為我們太過於專注在自然界的部分；在我們長大了的同時，也變得講求邏輯、理性。雖然我們可能會把自己能夠把看得到這些東西的能力關上，卻不代表我們的孩子也應該這麼做。事實上，他們不這麼做才比較關鍵。如果我們不去理會孩子們在床底下或是衣櫃裡所看到的「怪獸」，他們最後也就會把自己能夠看到這些東西的能力給關上，這不代表那些怪獸離開了，單純只是他們再也看不見而已。要是他們看不見這些怪獸，反倒會很容易受到這些邪靈的影響。

我曾經和一位媽媽和爸爸聊，他們的兒子無論是在家裡和學校都有行為偏差的問題。我們聊到有部分原因可能與睪固酮有關，後來也沒有再聊到其他的部分，不過等到下次我們又碰面時，他們倒是告訴了我一些新的消息。

有天晚上這個男孩在上床睡覺前先去上廁所，他媽媽走進來確認他的狀況，發現他在門邊上一直在哭。她一邊安慰他，一邊詢問發生了什麼事，他說自己看見了兩個邪靈，牠們小聲地在議論著他，因為牠們知道他看得見自己。後來兒子就告訴母親說，他發現原來就是因為這兩個邪靈，才會造成他表現不佳，但是他也說自己不想再聽牠們的話了。

後來這個男孩的行為就立刻改變了，他在家的表現開始不一樣，就連在學校也比之前表現得還要好，當他在學校的表現還沒有完全恢復正常的期間，他的父母就是一直持續地幫助他想起當時他自己領受到的啟示，我們都需要別人來提醒神曾經向我們顯明的事。

在發生這件事之前，我和他的父母都只一直在看一般而言撫養一個年幼男孩長大會遇到哪些挑戰，而沒有考慮到屬靈層面所可能帶來的影響。當父母們來找我幫忙時，我都會先從自然環境開始看起。我常覺得父母親太容易覺得自己的孩子一定是受到邪靈影響，反倒沒有先把一些自然會有的影響因素給排除掉。我們絕對不會希望明明沒有邪靈，但卻讓孩子誤以為自己是受到邪靈影響。我實在見過有太多人以為行為有問題就一定是邪靈在作祟，但其實可能最後發現不過是因為麩質或其他類似的過敏症所引起，如果這些自然界的事物都被排除掉了，那就很有必要看看在超自然的領域中是否有眼所不能見的東西在對這個情形造成影響。

你想想看，如果這個男孩被告知他所看到的東西不是真的，於是他就把自己能夠看見這些東西的能力給關上的話，他就永遠無法得知到底是什麼讓自己行為表現不佳了。要是沒有這樣的認知，他的行為大概就會很難有所改變。不過因為這個男孩看得見靈界的勢力在運行，而且他的父母相信他所說的，他們就立刻看見了改變。

同樣的道理反過來也會成立，如果有個孩子把自己能夠看見這些東西的能力關上，他/她的行為也可能會變本加厲。與其要孩子把能夠看見這些靈界事物的能力關掉，倒不如去幫助他們好好發展這項恩賜。

再次提問

能夠問對問題是經歷突破的主要關鍵；問題不光是能夠讓我們與孩子培養出良好的關係，問題也能夠教他們在面對挑戰時可以如何去徹底思考，並努力地去找到解套辦法。換句話說，藉由提問能夠賦予孩子們能力。有時候我們沒有意識到，如果我們只是一味地告訴他們事情是怎麼一回事，卻從來都不問他們怎麼看的話，其實是剝奪了他們解決問題的能力。

同樣地，當我們提出一些問題讓孩子們去問耶穌時，即是在幫助他們與耶穌建立關係；如果老是我們在問耶穌或是禱告，而沒有讓孩子自己這麼去做的話，可能會攔阻了他們與耶穌建立關係的機會。我發現要是一個孩子可以明白自己看到的那些嚇人東西到底是什麼，他們之後通常就不會再看到了。孩子們可以拿起自己在靈界裡的權柄。

看見是項需要被好好發展的恩賜

只不過有時候你和你的孩子也可能是面對到一場屬靈的長期抗戰，比方說你的女兒可能看見了一些可怕的東西，你就來教她怎麼奉耶穌的名命令牠們離開，牠們離開之後你的女兒就也可以再睡回去。不過隔天晚上這些東西又回來了，這時候不用擔心或是害怕，而是應該要視其為一個可以持續進行教育的大好良機。

神並不希望我們在遇到這樣的狀況時，光是想要找到可以解決問題的公式。因為我們太容易會認為，要是我們曾經有過一命令邪靈，牠們就離開的經驗，大概之後碰到這樣的情形就想要如法炮製。不過很不幸地，想要像這樣套公式卻違背了我們明明能看見的這回事。就好像耶穌是唯有當祂看見父在做什麼，祂才去做（見約翰福音五章19-20節），而且從所記錄下來的文字來看，祂從來沒有同一件事做兩次過。要是我們只做自己上次所用的辦法，那我們就沒有在運用自己能看見的能力，而只是憑記憶在行事（不過有時候神確實也會讓我們想起來上次曾經做過什麼，好讓我們知道這次該怎麼做。）

每次當邪靈彰顯的時候，就是一個機會讓孩子們可以加深與耶穌之間的關係。要是他們一直都只是上次怎麼做，這次就也照做的話，那他們就越來越不會去尋求耶穌。要是他們一直都只重複之前的做法，那麼他們就只是建立起一套套的公式，而沒有在建立與神的關係。

教導孩子知道該問什麼樣的問題，會有助於他們不淪為套用公式，而是會鼓勵他們知道要轉向耶穌。想像假設昨天晚上你的兒子在房間裡看到了可怕的東西，他命令牠們離開，牠們也確實離開了，不過現在牠們又出現了。

首先你可以告訴你的兒子說，他應該要拿起自己的屬靈遙控器，並且調到神的頻道。最基本的那一步一定都是要讓孩子可以把注意力重新對準神，有時候如果真的可以拿一個遙控器讓孩子在上面按一下按鈕會挺好的，因為那會有助於他們藉由身體的動作去更好聯想自己在屬靈上的操練。

接著你可以引導兒子去問下列這些問題：

耶穌在房間裡的哪裡？

祂穿什麼衣服？

祂手上有拿像武器那類的東西嗎？

祂有沒有帶任何東西呢？

祂為什麼要帶那樣東西？

祂拿那樣東西做什麼？

而這段對話有可能會像下面這樣進行：

孩子：祂穿著盔甲站在我的門邊，手上還拿著一把劍。祂沒有拿其他東西，不過祂把手上的劍給了我。我一拿過來，魔鬼就逃跑了，因為牠們不想被砍。

媽媽：你可以問耶穌那你現在拿著這把劍該做什麼。

孩子：祂叫我要把那把劍放在床底下，那把劍以後就是我的了。

隔天晚上可能又不一樣了。

孩子：祂今天戴著一頂很硬的帽子站在角落那邊，祂手上沒有拿任何武器，不過祂有一台壓路機，祂坐上去並且叫我跳到祂大腿上，一起開著壓路機把魔鬼們輾過去。

媽媽：就這麼做吧！

孩子：哈！牠們被壓成蔥油餅了！

不過某個晚上又再次改變了。

孩子：祂就在我身旁，祂穿著短褲和T-shirt，手上拿著一支網球拍。祂告訴我說：「你看好了！」祂衝過去把魔鬼們一個個都朝著黑黑的地方打過去了！

媽媽：聽起來好有趣喔！牠們都不見了嗎？

孩子：對啊，通通都不見了！

這個操練的要點就在於你的孩子必須要每次都從耶穌那裡得到答案，這個過程會加深他們與祂的關係，也會強化他們的心志，知道不應該只是上次哪招管用就用哪招，而是應該要每次都到耶穌面前讓祂來帶領和介入。

在剛剛所提到的第一個例子裡，孩子面臨到一場爭戰，而耶穌就給了他去應戰所需要的東西，祂也讓孩子看見魔鬼其實有多怕他。而在第二個情境裡，孩子是與耶穌一起合作去把魔鬼壓扁，在這個情境底下也比較沒有那麼嚴肅，而是多了些樂趣。壓路機通常是用來鋪路用的，因此可見祂渴望在這個孩子的裡面建造些什麼。而在最後的一個情境裡，是由耶穌出馬去為孩子做成，這會讓他知道祂也很享受在這個遊戲裡。孩子可以看見與耶穌相比，魔鬼實際有多小（就像顆網球一般大）。

這些畫面其實都挺有趣的，就像在玩遊戲一樣，而不是嚴肅的打仗，可是結果卻是一樣的：魔鬼最後都不見了。你的孩子與耶穌不會只能用某種正確的方式才能與耶穌相遇，很重要的一門功課是他們要學會問神，祂為什麼讓他們看見祂是怎樣的一位神以及那對他們而言意義為何。

### 🙏 禱告引導

問問看主，你生命中是否有哪裡的門是開著的，如果有的話，請祂為你把它關上，或是問祂你需要做什麼才能讓那道門被關上。問祂如果又有東西嘗試要去開那扇門的話，你該怎麼做好。把祂所告訴你的都寫下來。

問耶穌如果孩子告訴你說他們看到了一些東西，祂希望你怎麼做好，並去仔細聽看看每一個孩子是否有任何不同之處。花時間好好禱告，並想像你就是用耶穌所希望的方式去回應。當我與珍娜聊到她所經歷到的事情時，就發現惡夢是仇敵經常用來使這些很有恩賜的孩子感受到恐懼的一種主要方式。
