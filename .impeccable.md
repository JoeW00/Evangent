# Impeccable Design Context

## Design Context

### Users
- **華人教會家長**：華語教會的父母，想學習如何帶領孩子屬靈成長，需要中英對照理解原文
- **教會兒童事工同工**：兒童主日學老師、牧者，用於備課和教導參考
- **翻譯校對團隊**：負責中文翻譯的同工，需要中英並列比對以確保翻譯品質
- **相信 AI 傳福音的信徒**：在 AI 時代，相信神也會透過 AI 將光傳給世上的人
- **使用情境**：長時間閱讀、備課查考、翻譯校對，多為安靜獨處的場景

### Brand Personality
- **溫暖** (Warm)：如同一本溫暖的實體書，給人安心的閱讀體驗
- **寧靜** (Serene)：平靜、不喧鬧，適合靈修和深度閱讀
- **值得信賴** (Trustworthy)：可靠、專業、有品質感
- **情緒目標**：讀者應感到平安、被關懷、專注於內容本身

### Aesthetic Direction
- **視覺基調**：書卷氣質的數位閱讀器，介於傳統書籍與現代 web app 之間
- **色彩**：柔和暖色調（米白、淡金、暖灰），主色為沉穩藍灰 (#4F6D7A)，暖橘點綴 (#D4956A)
- **字體**：中文宋體 (Noto Serif TC) + 英文襯線 (Lora) + UI 無襯線 (Inter)
- **反面參考**：
  - 不要冰冷科技感（深色主題為主、霓虹色、SaaS 產品感）
  - 不要兒童卡通風（雖然主題與孩子有關，但讀者是成人，不要幼稚）
- **主題模式**：亮色模式為主，深色模式為輔（深色使用深靛藍而非純黑）

### Design Principles
1. **閱讀至上** (Reading First)：所有設計決策以長時間舒適閱讀為最高優先
2. **雙語平衡** (Bilingual Balance)：中英內容同等重要，透過微妙的視覺差異區分但不偏頗
3. **靜謐溫暖** (Quiet Warmth)：用留白和柔和色調營造寧靜氛圍，避免視覺噪音
4. **尊重內容** (Content Respect)：介面退居幕後，讓屬靈內容本身發光
5. **無障礙包容** (Accessible & Inclusive)：確保所有人都能使用，鍵盤導航、色彩對比、螢幕閱讀器支援
